## ✏️ Edit & PUSH from VS Code
1. **File → Open Folder** → pick the unzipped **GCS-clean** folder.
2. Edit a file, then **Save** (Ctrl+S).
3. Click **Source Control** in the left bar (the branch icon).
4. Type a short message (e.g. `update homepage headline`).
5. Click **✓ Commit**. If asked to stage all changes, click **Yes**.
6. Click **Sync Changes** (or the ⟳ icon, bottom-left).
   - **First time only:** it'll ask you to **sign in to GitHub** — do it. (This is the
     login I can't enter for you.)
7. Wait ~1 minute → the live site updates automatically.

## 🔄 Synchronize (pull the latest down)
- Source Control → **···** → **Pull**, or click the ⟳ icon at the bottom-left.
- Do this before editing if changes were made elsewhere.
